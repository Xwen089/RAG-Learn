### 新增组件：SessionManager
```python
class SessionManager:
    """会话管理器，负责会话元数据的持久化"""
    
    def load_sessions() -> Dict[str, Dict[str, Any]]:
        """从文件加载所有会话"""
    
    def save_sessions(sessions: Dict[str, Dict[str, Any]]):
        """保存所有会话到文件"""
    
    def create_session(name=None) -> Dict[str, Any]:
        """创建新会话"""
    
    def update_session(session_id: str, updates: Dict[str, Any], sessions: Dict[str, Dict[str, Any]]):
        """更新会话信息并保存"""
    
    def delete_session(session_id: str, sessions: Dict[str, Dict[str, Any]]) -> bool:
        """删除会话（包括元数据和历史文件）"""
    
    def get_all_sessions() -> Dict[str, Dict[str, Any]]:
        """获取所有会话（包括从文件系统发现的）"""
```

### 核心函数
1. `create_new_session()` - 创建新会话（自动持久化）
2. `switch_session(session_id)` - 切换会话（自动加载历史）
3. `rename_session(session_id, new_name)` - 重命名会话（自动持久化）
4. `delete_session(session_id)` - 删除会话（自动清理文件）
5. `get_current_messages()` - 获取当前会话消息（从内存或文件）
6. `update_current_messages(messages)` - 更新会话消息和信息（自动持久化）

### RAG集成
- 使用当前会话ID作为`session_id`参数
- 通过`file_history_store.get_history()`自动加载对话历史
- 使用`config.rag_prompt_template`作为提示词模板
- 对话历史自动保存到文件系统
- 每个会话有完全独立的上下文和历史

## 🔄 使用流程

1. **首次访问**：自动创建"新会话 1"
2. **新建会话**：点击"➕ 新建会话"创建新会话
3. **搜索会话**：在搜索框输入名称过滤会话
4. **重命名会话**：点击"⋯" → "✏️ 重命名"编辑名称
5. **删除会话**：点击"⋯" → "🗑️ 删除" → 确认删除
6. **切换会话**：点击会话卡片或切换按钮
7. **开始聊天**：在输入框中提问，AI基于当前会话历史回答
8. **清空会话**：点击"🗑️ 清空当前会话"重置对话

## 📊 数据存储

### 会话元数据持久化
- **存储位置**：`./chat_history/sessions.json`
- **存储内容**：会话ID、名称、创建时间、最后消息、消息数量等
- **自动加载**：应用启动时自动加载所有会话
- **实时保存**：创建、重命名、更新、删除会话时自动保存

### 对话历史持久化
- **存储位置**：`./chat_history/{session_id}`（每个会话一个文件）
- **存储内容**：完整的对话消息历史
- **技术实现**：通过`file_history_store`模块的`FileChatMessageHistory`类
- **自动同步**：RAG链自动保存和加载对话历史

### 数据恢复机制
1. **会话发现**：自动扫描`./chat_history/`目录，发现未记录的会话文件
2. **数据重建**：为发现的会话创建基本元数据记录
3. **历史加载**：切换会话时自动从文件加载对话历史
4. **错误处理**：文件损坏或格式错误时创建新的会话
